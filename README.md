# Flixter - A Video Streaming Platform (Teacher Courses for Students) <br />
https://flixter-karly.herokuapp.com <br />
This project is a two-sided video streaming marketplace platform. It features the capability to perform credit card transactions. Flixter involves user role management as well as complex user interfaces and advanced database relationships. It combines HTML, CSS, RUBY, RAILS, ALGORITHMS, and JAVASCRIPT. <br />
